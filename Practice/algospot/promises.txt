A.
Yes — both checks can be merged into a single DFS, because connectivity is just a 
special case of shortest path.

Your current logic:

connected_dfs() → check if end reachable
distance_dfs() → compute minimum distance

But notice:

If no path exists, the shortest distance is effectively ∞.
If a path exists, the shortest distance will be finite.

So one DFS that returns the minimum distance already tells you connectivity.

Intended solution: Floyd-Warshall algorithm
