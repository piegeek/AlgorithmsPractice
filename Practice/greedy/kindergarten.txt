A.
🧩 The 5 Common Greedy Archetypes
Almost all greedy problems fall into these:

1️⃣ Activity Selection Type
Key: earliest finish time
Reason: leaves maximum room for future

2️⃣ Resource Allocation Type
Key: best marginal gain per cost
Example: fractional knapsack

3️⃣ Gap Cutting Type
Key: pick largest/smallest adjacent gaps
Example: Kindergarten

4️⃣ Merge Smallest First
Key: combine smallest elements first
Example: Huffman coding

5️⃣ Stay Ahead Strategy
Key: keep partial solution always better than optimal’s prefix
Example: interval scheduling
